# Surveillance Patrol Test

## Objective
Validate surveillance drone movement and telemetry stability during patrol simulation.

## Validation Areas
- Camera feed consistency
- GPS telemetry updates
- Stable drone navigation
- Sensor topic availability

## Procedure
1. Launch Gazebo simulation
2. Start patrol workflow
3. Monitor telemetry topics
4. Verify camera feed output
5. Validate movement stability

## Expected Result
Drone should complete patrol route while maintaining stable telemetry and active surveillance feed.
